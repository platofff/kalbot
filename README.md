# kalbot

Бот с нескучными демотиваторами. В перспективе возможны бэкенды кроме вконтакте. Работает (на момент написания ридми) вот здесь: https://vk.com/kallinux

#### Функции:
- Демотиваторы из пересланных сообщений/картинок/полностью автоматические
- Споры для редактора http://objection.lol/maker из пересланных сообщений
- Другие мелочи
